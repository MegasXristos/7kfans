Development Guide

Building From Source

A step-by-step guide to compile Seven Kingdoms: Ancient Adversaries from the GPL source code.

📋

Introduction

This guide walks you through building Seven Kingdoms using GNU autotools, supporting most platforms provided you fulfil the dependencies below. For Ubuntu-specific notes, an older guide exists externally.

See also: System Requirements for running, and Download for the source tar file.

📦

Dependencies

Programs

  • make
  • GCC 4.6 or other C++11 compatible compiler

Required Libraries

  • SDL 2.0.8
  • enet 1.3.xx
  • OpenAL-soft

Optional Libraries

  • libcurl — for full 7kfans multiplayer integration
  • gettext 0.19 — for translation support
⚙️

Building the Game

Run the configure script:

$ ./configure

If you don't have configure, generate it using autogen.sh (only needed when pulling directly from source). Otherwise download a release tar.

Compile the source:

$ make

Install the game:

# make install

Run the game:

$ 7kaa
🕹️

Running Locally Without Installing

To run without make install, point to the game data folder using the SKDATA environment variable:

$ SKDATA=data ./src/7kaa
🎵

Installing the Music

First download the music package, then extract and copy the MUSIC folder into your data directory:

$ bzip2 -cd 7kaa-music-2.15.tar.bz2 | tar xvf -
$ mv 7kaa-music/MUSIC data/MUSIC
