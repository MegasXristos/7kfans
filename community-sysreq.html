System Requirements

Find out what you need to run Seven Kingdoms: Ancient Adversaries on your machine.

✅

To Play Seven Kingdoms: Ancient Adversaries

Any recent system with the following:

Operating System
Windows, Linux, or macOS
CPU
32-bit or 64-bit little-endian
Graphics
OpenGL or DirectX compatible
Network
IPv4 UDP for internet play
❌

Limitations

The following are not supported:

  • Big-endian CPU
  • IPv6 connection
  • Localized right-to-left message rendering
  • Non-US keyboard mapping and UTF-8 input
⬇️

Ready to Install?

If your system meets the requirements above, head over to the download page and get the latest version.
